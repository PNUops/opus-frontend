<h1 align="center">부산대학교 SW프로젝트관리시스템 OPUS</h1>

<div align="center">
 <b>Opus</b>는 ‘작품’을 의미하는 영어 단어로,<br>
 부산대학교 내 소프트웨어 프로젝트(캡스톤/해커톤/교과 등)의 성과를 등록, 관리, 공유할 수 있는 시스템입니다.<br>
 또한, 운영 측면에서 대회, 팀, 결과물을 효율적으로 관리할 수 있도록 설계되었습니다.<br>
